# Tarea 1 - Inteligencia Artificial 2026-01
## Objetivo de la tarea:

Parte 1: Aprender estructura de red Bayesiana asociada al dataset "Mushroom", a través de los métodos de Hill-Climbing y Exhaustive Search. Ademas de aumentar el tamaño del dataset con datos sintéticos.  

Parte 2: Estudiar dataset "Human Activity Recognition Using Smartphones" para construir un HMM discretizado para luego realizar inferencias dado una secuencia de observaciones.

## Datasets
Parte 1:  
https://archive.ics.uci.edu/dataset/73/mushroom  
Parte 2:  
https://archive.ics.uci.edu/dataset/240/human+activity+recognition+using+smartphones
